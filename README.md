# Personal Portfolio

A sleek, minimalistic, and high-performance personal portfolio and blog. 

Built on the philosophy of **"Zero-JS by Default"**, it prioritizes accessibility, speed, and modern typography using Astro's static site generation capabilities and Tailwind CSS v4 for a seamless design system.

## Version
* **Current Version:** 1.0.0
* **Status:** Initial Development

## Tech Stack & Versions
* **Framework:** [Astro](https://astro.build/) (v5.0+)
* **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (`tailwindcss@^4.0.0`, `@tailwindcss/vite`)
* **Environment:** Node.js (v25+) and npm (v11+)
* **Icons:** [Lucide Icons](https://lucide.dev/)
* **Typography:** Inter

## Prerequisites
Before you begin, ensure you have the following installed:
* [Node.js](https://nodejs.org/) (v18 or higher recommended)
* npm (comes with Node.js)

## Installation

1. Clone or ensure you are in the project directory:
   ```bash
   cd portfolio
   ```

2. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Running the Project Locally

To start the local development server:

```bash
npm run dev
```

Your portfolio will now be running on `http://localhost:4321`.

### Other Commands
* `npm run build`: Builds the static site for production into the `dist/` logic.
* `npm run preview`: Previews your production build locally.
